### Automatic Pet Feeder (LabVIEW Simulation)

A LabVIEW-based simulation of an automated pet feeder, demonstrating control logic and user interface design.

**Key Features**
- LabVIEW front panel simulates an automatic feeding system for household pets.
- Block diagram demonstrates the logic for monitoring food levels and timer-based dispensing.
- Includes visual components for storage unit, food bowl, and timer control.

**Technologies Used**
- **Software:** LabVIEW  
- **Components Simulated:** Sensors, storage unit, timer control  
- **Functionality:** Simulation of automated food dispensing logic
